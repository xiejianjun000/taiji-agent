> 本文档为外部开发团队的任务指引。开发团队须按顺序完成「克隆→梳理→分析→设计→开发→交付」全流程，每个阶段须经验收方审核通过后方可进入下一阶段。

## 零、项目概述

### 0.1 项目名称

**Taiji Agent**（太极智能体）

### 0.2 一句话定位

政务场景千人级自进化智能体系统，本地优先，三层架构（运行时+引擎+灵魂），越用越懂业务。

### 0.3 北极星目标

> **"人人都有智能体，越用越好懂越好用，越来越懂业务。"**

| 用户语言 | 技术实现 |
|----------|----------|
| 人人都要有智能体 | Harness 并发 + Hermes 多租户 = 千人千面 Agent 网络 |
| 越用越好懂 | Hermes 跨会话记忆 + 个体进化层 |
| 越用越好用 | Hermes 三层进化（个人→部门→系统），共性沉淀 |
| 越来越懂业务 | Taiji Verify 防虚幻 + 知识库注入 + GovMCP 政务合规 |

### 0.4 验收方与开发方

| 角色 | 职责 |
|------|------|
| **验收方（咔咔）** | 只审只评，不写代码；审核分析报告、设计文档、里程碑交付物 |
| **开发团队** | 按任务书要求执行；完成克隆、梳理、分析、设计、开发、交付 |
