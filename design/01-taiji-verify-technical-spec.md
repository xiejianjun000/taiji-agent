(d 维) cos(I, G) = (I·G) / (‖I‖·‖G‖) ΔS ∈ [0, 1] ``` **锚点扩展版本**（当命名实体标注可用时）: ``` sim_est = w_e · mean(cos(I_e, G_e)) + w_r · mean(cos(I_r, G_r)) + w_c · mean(cos(I_c, G_c)) ΔS = 1 - clip(sim_est, 0.0, 1.0) 默认权重: w_e=0.5, w_r=0.3, w_c=0.2 ``` #### 3.1.2 阈值体系 | 闸区 | ΔS 范围 | 颜色 | 含义 | 触发动作 | |------|---------|------|------|----------| | Safe (安全) | [0, 0.30) | 🟢 | 语义对齐良好 | 正常推理，可输出 | | Transit (过渡) | [0.30, 0.50) | 🟡 | 轻度漂移 | 检查 λ 观察值，加强巽调 | | Risk (风险) | [0.50, 0.70) | 🟠 | 语义张力过大 | 强制记录语义节点，触发坤守残差修正 | | Danger (危险) | [0.70, 1.0] | 🔴 | 语义崩溃风险 | 立即触发复归(Fu Return) | > **注意**: 算法设计草案中的阈值 (0.40/0.60/0.85) 与本规格书的阈值 (0.30/0.50/0.70) 存在差异。本规格书采用更为严格的安全阈值，以适配政务场景对高一致性的要求。实际部署时可通过配置切换。 #### 3.1.3 数据结构 ```python @dataclass class DeltaSResult: delta_s: float # 阴阳距 [0, 1] similarity_raw: float # 原始余弦相似度 zone: str # "safe" | "transit" | "risk" | "danger" zone_enum: int # 0 | 1 | 2 | 3 sim_est: Optional[float] # 锚点扩展相似度（如有锚点） anchor_breakdown: Optional[Dict[str, float]] # 各锚点类型得分 @dataclass class AnchorSet: entities: List[Tuple[np.ndarray, np.ndarray]] # [(I_e, G_e), ...] relations: List[Tuple[np.ndarray, np.ndarray]] # 关系锚点对 constraints: List[Tuple[np.ndarray, np.ndarray]] # 约束锚点对 ``` #### 3.1.4 Python 伪代码 ```python import numpy as np from dataclasses import dataclass from typing import List, Tuple, Optional, Dict def cosine_similarity(a: np.ndarray, b: np.ndarray) -> float: """计算余弦相似度""" dot = np.dot(a, b) norm = np.linalg.norm(a) * np.linalg.norm(b) return float(dot / norm) if norm > 0 else 0.0 def calculate_delta_s( I: np.ndarray, G: np.ndarray, anchors: Optional[AnchorSet] = None, weights: Optional[Dict[str, float]] = None, ) -> DeltaSResult: # Step 1: 基础余弦相似度 sim_raw = cosine_similarity(I, G) if anchors is not None: # Step 2: 锚点扩展 w = weights or {"entities": 0.5, "relations": 0.3, "constraints": 0.2} sim_entities = np.mean([cosine_similarity(e[0], e[1]) for e in anchors.entities]) if anchors.entities else 0.0 sim_relations = np.mean([cosine_similarity(r[0], r[1]) for r in anchors.relations]) if anchors.relations else 0.0 sim_constraints = np.mean([cosine_similarity(c[0], c[1]) for c in anchors.constraints]) if anchors.constraints else 0.0 sim_est = w["entities"] * sim_entities + w["relations"] * sim_relations + w["constraints"] * sim_constraints sim_est = np.clip(sim_est, 0.0, 1.0) delta_s = 1.0 - float(sim_est) else: delta_s = 1.0 - sim_raw sim_est = None # Step 3: 确定闸区 if delta_s < 0.30: zone, zone_enum = "safe", 0 elif delta_s < 0.50: zone, zone_enum = "transit", 1 elif delta_s < 0.70: zone, zone_enum = "risk", 2 else: zone, zone_enum = "danger", 3 return DeltaSResult( delta_s=delta_s, similarity_raw=sim_raw, zone=zone, zone_enum=zone_enum, sim_est=sim_est, ) ``` ### 3.2 坤守 (Kun Guard /
