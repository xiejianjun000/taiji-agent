# Taiji Agent Console UI 预览 v2.0

**文件位置**: `/workspace/taiji-agent/ui.html`
**设计风格**: 太极道家 · 道法自然 · 暗夜星河

---

## 设计理念

采用**太极道家**设计风格，融合东方智慧与现代科技：
- 深色背景配以金色点缀，营造神秘高端感
- 太极八卦元素贯穿始终
- 道家典籍引用增添文化底蕴
- 流畅动画展现阴阳流转之理

## 技术特点

| 特性 | 说明 |
|------|------|
| **视觉风格** | 暗夜星河 + 金色点缀 + 太极八卦 |
| **字体** | ZCOOL 小微 + Noto Serif SC |
| **动画** | 太极旋转、粒子漂浮、卡片悬停 |
| **配色** | 玄青 #1a1a2e / 象牙 #f8f4e8 / 金色 #c9a962 |
| **响应式** | 支持桌面端、平板、移动端 |

---

## 启动方式

```bash
cd /workspace/taiji-agent
python -m http.server 8000
# 浏览器打开 http://localhost:8000/ui.html
```
