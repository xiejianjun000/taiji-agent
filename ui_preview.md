## 功能模块

| 模块 | 图标 | 功能 |
|------|------|------|
| **控制台** | 📊 | 系统状态监控、统计数据、三层架构状态 |
| **对话** | 💬 | AI 聊天界面，支持发送消息 |
| **记忆库** | 📚 | 跨会话记忆管理 |
| **审批工作流** | 📋 | 待审批请求列表 |
| **系统设置** | ⚙️ | Agent 配置参数 |

---

## 技术特点

- **深色主题**: 现代化深色 UI 设计
- **侧边导航**: 固定左侧导航栏
- **响应式卡片**: 圆角卡片布局
- **状态指示**: 运行状态实时显示
- **动画效果**: 悬停效果、状态脉冲动画

---

## 启动方式

```bash
cd /workspace/taiji-agent
python -m http.server 8000
```

然后在浏览器中打开: `http://localhost:8000/ui.html`
